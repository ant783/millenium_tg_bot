# 🐱‍💻 Millenium Proxy Bot — Прокси для Telegram без нервов! 🚀

"Когда Telegram говорит 'заблокировано', а ты отвечаешь 'держи прокси!' 🤫

# 😎 Что это за зверь?
Millenium — это Telegram-бот, который молниеносно выдаёт свежие MTProto-прокси с 4 надёжных источников. Нажимаешь кнопку — получаешь прокси с кнопками "connect". Всё. Работает даже если пол-интернета лёг.

# 🎮 Как пользоваться (даже бабушка разберётся)
Запусти /start — получишь кнопку
Жми "🔄 Обновить прокси" — бот ищет прокси
Нажимай "connect" под каждым — Telegram сам подключится
